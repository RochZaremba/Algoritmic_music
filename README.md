# 🎵 Algorithmic Music — Sonic Pi Projects

This repository contains my assignments, exercises, and creative experiments completed for the **Algorithmic Music** course using **Sonic Pi**.

## About

The goal of this repository is to collect and organize code written during classes and at home as part of the course. The projects focus on creating music through programming, exploring sound synthesis, and experimenting with rhythm, melody, texture, and live coding techniques.

## What you can find here

This repository may include:

- class assignments
- homework tasks
- short musical compositions
- Sonic Pi experiments
- studies of sound synthesis and timbre
- live coding practice

Each folder or file usually represents a separate task, exercise, or mini-project.

## Technologies

This repository is based on:

- **Sonic Pi**
- its Ruby-like live coding language
- basic concepts of algorithmic composition
- sound synthesis techniques
